# Android AR Ruler (安卓测距仪)

这是一个基于 Google ARCore 框架开发的安卓增强现实（AR）测距仪应用。通过手机相机识别环境平面，实现高精度的长度测量。

## 🌟 功能特点

- **平面检测**：支持水平面和垂直面的自动识别。
- **高精度测量**：通过 ARCore 空间定位，实现厘米级的距离测算。
- **实时反馈**：移动过程中实时显示当前点到上一个锚点的距离。
- **多点连线**：支持连续添加多个测量点，自动计算各段距离。
- **ARCore 离线安装**：内置 ARCore 框架，支持离线安装。
- **现代 UI 设计**：使用 Jetpack Compose 构建，界面简洁、响应迅速。

## 🛠️ 技术栈

- **开发语言**：Kotlin
- **界面框架**：Jetpack Compose
- **AR 引擎**：Google ARCore
- **图形渲染**：OpenGL ES 2.0 (用于摄像头背景及引导点渲染)
- **架构**：遵循现代 Android 开发最佳实践

## 🚀 快速开始

### 前置条件
- Android 手机（需支持 ARCore）。
- 授予相机权限。
- 打开应用后，若未安装ARCore会提示安装，安装完毕后请重启应用
- 若卡在初始化界面，请重启应用
- 反正就是万事不决就重启

### 使用说明
1. **初始化**：打开应用后，缓慢移动手机以识别周围环境（地面或墙面）。
2. **锁定平面**：当屏幕中央出现绿色准星或网格点时，表示已成功捕捉到平面。
3. **添加测量点**：点击屏幕下方的 `+` 按钮，在当前准星位置放置一个锚点。
4. **测量距离**：移动手机到下一个位置，应用将实时显示起始点到当前位置的距离。再次点击 `+` 按钮固定测量线。
5. **清除数据**：点击左下角的删除图标即可清空所有测量点。
